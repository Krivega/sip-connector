# События `StatsPeerConnection`

`StatsPeerConnection` генерирует события при сборе WebRTC статистики. Все события доступны через префикс `stats:*` в `SipConnector`.

## События

| Имя события       | Описание                                 | Тип данных                                             |
| ----------------- | ---------------------------------------- | ------------------------------------------------------ |
| `stats:collected` | Генерируется при сборе статистики WebRTC | `{ outbound: TOutboundStats; inbound: TInboundStats }` |

## Структуры данных

### `TInboundStats`

Статистика входящих медиа-потоков, включающая аудио и видео потоки, кодеки, ICE кандидаты и транспортную информацию.

### `TOutboundStats`

Статистика исходящих медиа-потоков, включающая аудио и видео потоки, кодеки, ICE кандидаты и транспортную информацию.

Подробное описание структуры статистики доступно в [рецепте "WebRTC Статистика"](../recipes/webrtc-stats.md).
