# Документация sip-connector

## Разделы документации

### [Рецепты](./recipes/README.md)

Практические примеры и руководства по использованию библиотеки. Все рецепты содержат готовые примеры кода и подробные объяснения.

**Содержание:**

- Начало работы (базовый пример, работа с сессией)
- Звонки и медиа (входящие звонки, медиа-потоки, презентации, ICE restart)
- Конференции и участники
- События и мониторинг
- Оптимизация и производительность
- Надежность и восстановление
- Рекомендации и отладка

### [API Reference](./api/README.md)

Справочная документация по API библиотеки, включающая все классы, методы, утилиты, типы и события.

**Содержание:**

- API и экспорты
- События всех менеджеров:
  - ConnectionManager
  - CallManager
  - IncomingCallManager
  - PresentationManager
  - StatsPeerConnection
  - VideoSendingBalancerManager
  - MainStreamHealthMonitor
  - SessionManager
  - AutoConnectorManager
  - ApiManager
  - SipConnector

### [Архитектура](./architecture/README.md)

Детальное описание внутренней структуры библиотеки, принципов проектирования и архитектурных решений.

**Содержание:**

- Обзор архитектуры
- Компоненты (все менеджеры)
- Модель состояний (XState)
- Архитектурные принципы
